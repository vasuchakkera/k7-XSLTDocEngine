    Function : functx:mmddyyyy-to-date

    Information about this function

    Documentation for functx:mmddyyyy-to-date

    This function changes the date from mmddyyyy to a date
    Examples
    • functx:mmddyyyy-to-date('12-15-2004') results in 2004-12-15
    • functx:mmddyyyy-to-date('12152004') results in 2004-12-15
    • functx:mmddyyyy-to-date('12/15/2004') results in 2004-12-15
    History:
    Some background information for this function. In the input XML there are three indicators namely
    • displaydate
    • displayday
    • displaymonth
    When these three indicators are set to 'yes' , only then we should be returning a value. if any of these have a value other than 'yes' then this function returns null. This functionality was introduced in the Sprint 13 , When a new rule was rolled out, and dictated that these dates should only be populated when the clients want them to be.
